# 3DIP-SGST
Code for paper: **SGST-Transformer: A Spherical Geometry-Aware Spatio-Temporal Transformer for 360° Video Saliency Prediction**

## 📁 Experimental Results
We provide the predicted saliency maps and quantitative results on the following datasets:
- sports360
- AVS-ODV
- SVGC-AVA
- 360AV-HM
